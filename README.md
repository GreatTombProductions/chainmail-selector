# ⛓️ Chainmail Ring Selector

Interactive calculator for finding the right ring sizes for chainmail weaving projects.

## What It Does

Enter a weave pattern (European 4-in-1, Byzantine, etc.) and see which wire gauge + inner diameter combinations will work. The calculator:

- Shows aspect ratios (AR) for all ring combinations in a visual grid
- Highlights ideal matches and compatible ranges for your selected weave
- Supports 20+ weave patterns with different AR requirements
- Handles two-size weaves (like Helm Chain and Dragonscale) that need primary + secondary rings
